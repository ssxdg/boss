# Boss 直聘辅助投递浏览器扩展

这是一个 Chrome/Edge Manifest V3 扩展，用于在用户已登录 Boss 直聘网页版后，根据本地配置辅助筛选岗位、填充沟通模板并记录结果。

## 合规边界

- 扩展不处理登录流程，不保存账号、密码或登录凭据。
- 扩展不绕过验证码、人机验证、安全校验或平台风控。
- 扩展不逆向、不破解、不调用未授权接口批量抓取数据。
- 遇到未登录、验证码、安全验证、页面结构异常、每日上限或用户暂停时会停止自动化。

## 功能

- 配置岗位关键词、城市、薪资上下限、每日投递上限和随机间隔。
- 编辑多条沟通模板，支持 `{jobTitle}`、`{company}`、`{city}`、`{salary}` 变量。
- 启动、暂停、查看运行状态和今日投递进度。
- 查看、清空投递日志。
- 导入、导出 JSON 配置。
- 使用 `chrome.storage.local` 保存本地配置、状态和日志。

## 开发命令

```bash
npm install
npm run typecheck
npm test
npm run build
```

## 加载扩展

1. 执行 `npm run build`。
2. 打开 Chrome 或 Edge 的扩展管理页。
3. 开启开发者模式。
4. 选择“加载已解压的扩展程序”。
5. 选择本项目的 `dist` 目录。

## 使用方式

1. 打开 Boss 直聘网页版并保持已登录。
2. 打开扩展弹窗，填写关键词、城市、薪资、上限、随机间隔和沟通模板。
3. 点击“保存”，再点击“启动”。
4. 如页面出现验证码、安全验证、未登录或结构异常，扩展会暂停并在状态区和日志中说明原因。

## 构建产物

构建后的 `dist` 目录包含：

- `manifest.json`
- `background.js`
- `content.js`
- `index.html`
- 前端资源文件

该目录可直接作为 Chrome/Edge 未打包扩展加载。
